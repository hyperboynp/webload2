### 4. Menu Management
**Path**: `Appearance â†’ Menus`

**Menu Locations**:
- **Primary Menu** - Main navigation in header
- **Footer Menu** - Links in footer bottom (Privacy Policy, Terms, etc.)

**How to Create Menus**:
1. Click "Create a new menu"
2. Give it a name (e.g., "Primary Menu")
3. Add items from left panel (Pages, Custom Links, Categories)
4. Drag items slightly right to create dropdown/submenu
5. Assign to menu location (Primary Menu or Footer Menu)
6. Click "Save Menu"

## ğŸ†˜ Getting Unstuck

### Can't Find a Setting?
1. Check this guide's section list
2. Use browser search (Ctrl+F) on this page
3. Try searching in Customizer search box
4. Check "Additional Settings" in sidebar

### Changes Not Showing?
1. Clear browser cache (Ctrl+F5)
2. Check if section is enabled in Homepage Settings
3. Ensure you clicked "Publish" not just "Save Draft"
4. Check if you're on the correct page

### Lost Changes?
1. Check "Revisions" in page/post editor (right sidebar)
2. Use "Revert" in Customizer if you haven't closed it
3. Contact your hosting provider for backup restoration
